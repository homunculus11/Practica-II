# Practica II Console App

Standalone console version of the course-management app.

## Run

From the project root:

```bat
cd C:\Users\PC\Desktop\Practica-II\Java\ConsoleApp
.\run.bat
```

PowerShell:

```powershell
cd C:\Users\PC\Desktop\Practica-II\Java\ConsoleApp
.\run.ps1
```

## Database

This app has its own database config:

```text
Java\ConsoleApp\resources\db.properties
```

It also has its own copies of the SQL Server JDBC jar and Windows authentication DLL under:

```text
Java\ConsoleApp\lib
```
